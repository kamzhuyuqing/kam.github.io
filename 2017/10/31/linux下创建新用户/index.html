linux下创建新用户

发表于 2017-10-31 | 分类于 linux

由于优惠季买了两年阿里的服务器，OS是Ubuntu14.04。今天就来配置一下服务器用户。

用户增加

adduser 命令

adduser 会默认自动在home目录下面创建用户目录，自动使用系统shell脚本，在创建的时候输入用户密码。
示例：创建 testapp 用户





useradd 命令

useradd 相比之下没有 adduser那么‘智能’， 不会自动创建用户目录， 需要手动指定用户所使用的脚本命令目录，需要手动创建用户的密码。

常用命令行选项：

  • （1） -d： 指定用户的主目录
  • （2） -m： 如果存在不再创建，但是此目录并不属于新创建用户；如果主目录不存在，则强制创建； -m和-d一块使用。
  • （3） -s： 指定用户登录时的shell版本
  • （4） -M： 不创建主目录

示例：创建用户testapp2





用户删除

userdel 命令

示例：删除testapp2





如果要完全删除用户信息，包括遗留下来的文件夹等信息，可以加上 -r 参数。
